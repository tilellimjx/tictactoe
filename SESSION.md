# Session Context

Last updated: 2026-04-28

## What was built

A multi-agent requirements pipeline for a Python CLI Tic Tac Toe game.

## File layout

```
agent_utils.py                  # Shared tools + agent loop (imported by all agents)

requirements_agent.py           # Requirements agent — Opus 4.7
requirements_agent_sonnet.py    # Requirements agent — Sonnet 4.6 + adaptive thinking
cross_review_agents.py          # 3-step cross-review + merge pipeline
coding_agent.py                 # Implements game.py from requirements_final.md
unit_testing_agent.py           # Writes and runs test_game.py against game.py

requirements/
  requirements_opus.md          # Agent 1 output
  requirements_sonnet.md        # Agent 2 output
  requirements_opus_merged.md   # Opus's review of Sonnet's doc
  requirements_sonnet_merged.md # Sonnet's review of Opus's doc
  requirements_final.md         # Definitive final spec

game.py                         # Generated by coding_agent.py
test_game.py                    # Generated by unit_testing_agent.py
scores.json                     # Created at runtime by game.py (gitignored)
```

## Environment variables (all agents)

| Variable           | Required | Default                              | Description                          |
|--------------------|----------|--------------------------------------|--------------------------------------|
| ANTHROPIC_API_KEY  | Yes      | —                                    | Anthropic API key                    |
| WORK_DIR           | No       | Script directory                     | Project root override                |
| AGENT_ANSWERS      | No       | (prompts stdin)                      | Newline-separated answers for requirements agents (remote mode) |
| REQUIREMENTS_FILE  | No       | requirements/requirements_opus.md    | Requirements agent output path       |
| OUTPUT_FILE        | No       | game.py                              | Coding agent output path             |
| GAME_FILE          | No       | game.py                              | Unit testing agent input path        |
| TEST_FILE          | No       | test_game.py                         | Unit testing agent output path       |
| MAX_ITERATIONS     | No       | 30                                   | Agent loop cap                       |

## How to run

```bash
export ANTHROPIC_API_KEY="sk-ant-api03-..."

# Requirements agents — remote/non-interactive mode:
export AGENT_ANSWERS="Command-line terminal (Python)\nBoth Human vs Human and Human vs AI, with three AI difficulty levels: Easy, Medium, and Hard\nScore tracking across rounds that persists to a file so scores are saved when the app closes, and a replay option after each game"
python requirements_agent.py
python requirements_agent_sonnet.py

# Cross-review + final merge:
python cross_review_agents.py

# Implement the game from requirements_final.md:
python coding_agent.py

# Write and run the test suite:
python unit_testing_agent.py
```

## Agreed requirements scope

- **Platform**: Python command-line terminal
- **Game modes**: Human vs Human AND Human vs AI
- **AI difficulty**: Easy, Medium, Hard
- **Score tracking**: persisted to file across sessions
- **Extra**: replay option after each game
